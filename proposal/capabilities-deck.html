Requirement R2

Restocking
Recommendations

What it does

Combines current stock levels + demand forecast + operator-supplied budget ceiling to generate ranked purchase order recommendations per warehouse and category.

How we build it

New FastAPI endpoint for recommendation logic. New Vue view consistent with existing design system. Budget ceiling as an adjustable input — operations team stays in control.

This is the feature R. Tanaka's team has been waiting for. We'll validate the recommendation logic with her team before shipping.
